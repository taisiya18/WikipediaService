# Каталог фильмов

## Описание проекта

Проект представляет собой консольное приложение для управления каталогом фильмов. Программа позволяет:
- Просматривать список фильмов.
- Добавлять новые фильмы.
- Редактировать рейтинг фильмов.
- Удалять фильмы.
- Получать рекомендации на основе любимых жанров.
- Добавлять фильмы с автоматическим заполнением данных из Wikipedia.
- Сохранять изменения в файл.

## Возможности программы

### Основные функции
1. **Просмотр всех фильмов:**
   - Выводит список всех фильмов в каталоге с их названием, жанром, годом выпуска и рейтингом.

2. **Добавление фильма:**
   - Пользователь может вручную добавить фильм, указав его название, жанры, год выпуска и рейтинг.

3. **Редактирование рейтинга фильма:**
   - Пользователь может изменить рейтинг фильма, выбрав его по названию.

4. **Удаление фильма:**
   - Пользователь может удалить фильм из каталога, выбрав его по названию.

5. **Просмотр фильмов в таблице:**
   - Фильмы отображаются в виде таблицы с использованием библиотеки Spectre.Console.

6. **Диаграмма по жанрам:**
   - Программа строит диаграмму, показывающую распределение фильмов по жанрам.

7. **Рекомендации:**
   - Пользователь может ввести свои любимые жанры, и программа предложит фильмы из каталога, соответствующие этим жанрам.

8. **Добавление фильма из Wikipedia:**
   - Пользователь может добавить фильм, указав его название. Программа автоматически попытается найти информацию о фильме (название, жанр, год выпуска) в Wikipedia.

9. **Сохранение данных:**
   - При завершении работы программа сохраняет все изменения в файл.

### Дополнительные функции
- **Поиск данных из Wikipedia:**
  - Программа использует API Wikipedia для поиска информации о фильме. Если фильм найден, программа предлагает добавить его в каталог с автоматически заполненными данными.
  - Если информация о жанре или годе выпуска отсутствует, пользователь может ввести её вручную.

- **Обработка ошибок:**
  - Программа корректно обрабатывает отсутствие файла с фильмами, предлагая пользователю либо указать путь к файлу, либо начать с пустого каталога.

## Как работает программа

### Запуск программы
1. Программа запрашивает у пользователя, находится ли файл `movies.txt` в папке проекта.
   - Если пользователь отвечает `y`, программа ищет файл в текущей рабочей директории.
   - Если пользователь отвечает `n`, программа запрашивает полный путь к файлу.

2. Если файл найден, программа загружает данные о фильмах. Если файл не найден, программа начинает работу с пустого каталога.

### Основной цикл работы
1. Программа отображает меню с доступными опциями.
2. Пользователь выбирает опцию, вводя соответствующую цифру.
3. В зависимости от выбора пользователя, программа выполняет одно из действий:
   - Просмотр фильмов.
   - Добавление фильма (вручную или из Wikipedia).
   - Редактирование рейтинга.
   - Удаление фильма.
   - Просмотр таблицы фильмов.
   - Построение диаграммы по жанрам.
   - Получение рекомендаций.
   - Выход и сохранение данных.

### Поиск данных из Wikipedia
1. Пользователь выбирает опцию "Добавить фильм из Wikipedia".
2. Программа запрашивает название фильма.
3. Используя API Wikipedia, программа ищет информацию о фильме:
   - Название.
   - Жанр.
   - Год выпуска.
4. Если информация найдена, программа предлагает добавить фильм в каталог.
5. Если информация о жанре или годе выпуска отсутствует, пользователь может ввести её вручную.

### Сохранение данных
1. При завершении работы программа сохраняет все изменения в файл `movies.txt`.
2. Файл сохраняется в формате:
Название фильма|Жанр1,Жанр2|Год выпуска|Рейтинг

## Требования

- .NET 6.0 или выше.
- Библиотека Spectre.Console для визуализации таблиц и диаграмм.
- Доступ к интернету для поиска данных в Wikipedia.

## Пример использования

- Запустите программу через dotnet run
- Укажите путь к файлу movies.txt или начните с пустого каталога (если копируете полностью проект, то там уже лежит файл - нужно лишь написать в консоли movies.txt)
- Используйте меню для управления каталогом фильмов:
- Добавьте фильм вручную или из Wikipedia.
- Просмотрите список фильмов.
- Отредактируйте рейтинг фильма.
- Удалите фильм.
- Получите рекомендации.
- Завершите работу программы, чтобы сохранить изменения.
